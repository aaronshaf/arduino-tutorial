LED Blink

Learn how to make an LED turn on and off. This is the "Hello World" of Arduino!

What You'll Learn

  • How to build a basic circuit
  • How breadboards work
  • How to write Arduino code
  • How to upload code to your Arduino

What You Need

Parts:

Arduino R3 board

1 LED (any color)

1 resistor (220 ohm)

Breadboard

2 jumper wires
