Write the Code

Now let's write the program to control our LED:

// LED Blink Program
// The LED is connected to pin 13

void setup() {
  // Tell Arduino that pin 13 is an output
  pinMode(13, OUTPUT);
}

void loop() {
  // Turn the LED on
  digitalWrite(13, HIGH);
  
  // Wait for 1 second (1000 milliseconds)
  delay(1000);
  
  // Turn the LED off
  digitalWrite(13, LOW);
  
  // Wait for 1 second
  delay(1000);
}

Understanding the Code:

void setup()
Runs once when Arduino starts. We use it to set up pin 13 as an output.
pinMode(13, OUTPUT)
Tells Arduino that pin 13 will send signals OUT (not receive them).
void loop()
Runs over and over forever. This is where our blinking happens.
digitalWrite(13, HIGH)
Sends 5V to pin 13, turning the LED ON.
digitalWrite(13, LOW)
Sends 0V to pin 13, turning the LED OFF.
delay(1000)
Pauses for 1000 milliseconds (1 second).
Try this: Change the delay values to make the LED blink faster or slower. For example, try delay(500) for half a second, or delay(2000) for 2 seconds.
