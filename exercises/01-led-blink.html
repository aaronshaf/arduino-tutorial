Exercise 1: LED Blink

Learn how to make an LED turn on and off. This is the "Hello World" of Arduino!

1

What You Need

Parts:

Arduino R3 board

1 LED (any color)

1 resistor (220 ohm)

Breadboard

2 jumper wires

2

How a Breadboard Works

A breadboard lets you connect parts without soldering. Here's how it works:

Simple Rules:

  1. Holes in same row = connected
  2. Different rows = not connected
  3. Center gap = keeps left and right separate
Remember: Holes in the same row are connected. The gap in the middle separates the two sides!
3

Place the LED

First, let's put the LED on the breadboard:

✓ Look at your LED - one leg is longer than the other

✓ The long leg is positive (+)

✓ The short leg is negative (-)

✓ Push both legs into the breadboard in different rows

4

Connect the Power (Red Wire)

Connect Arduino Pin 13 to the LED's long leg:

✓ Red wire from Pin 13 on Arduino

✓ Other end in same row as LED's long leg (+)

5

Add the Resistor

Connect the resistor to the LED's short leg:

✓ One end in same row as LED's short leg (-)

✓ Other end in a different row

✓ Direction doesn't matter for resistors

6

Complete the Circuit (Black Wire)

Connect Arduino GND to complete the circuit:

✓ Black wire from GND on Arduino

✓ Other end in same row as resistor's free end

✓ You should see current flowing in a complete loop!

Done! Your circuit is complete. The electricity flows: Pin 13 → LED → Resistor → GND
7

Write the Code

Copy this code into your Arduino IDE:

// LED Blink Program
// The LED is connected to pin 13

void setup() {
  // Tell Arduino that pin 13 is an output
  pinMode(13, OUTPUT);
}

void loop() {
  // Turn the LED on
  digitalWrite(13, HIGH);
  
  // Wait for 1 second (1000 milliseconds)
  delay(1000);
  
  // Turn the LED off
  digitalWrite(13, LOW);
  
  // Wait for 1 second
  delay(1000);
}
8

Upload and Test

  1. Connect your Arduino to your computer with the USB cable
  2. Open the Arduino IDE
  3. Select your Arduino board (Tools → Board → Arduino Uno)
  4. Select the right port (Tools → Port)
  5. Click the Upload button (arrow pointing right)
  6. Watch your LED blink!

9

How It Works

pinMode(13, OUTPUT) - This tells Arduino that pin 13 will send out electricity

digitalWrite(13, HIGH) - This sends electricity to pin 13, turning the LED on

digitalWrite(13, LOW) - This stops electricity to pin 13, turning the LED off

delay(1000) - This makes Arduino wait for 1000 milliseconds (1 second)
