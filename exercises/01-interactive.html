Try It Yourself!

See how changing the delay time affects the blinking speed:

Your Code Would Look Like:

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);   // LED on time
  
  digitalWrite(13, LOW);
  delay(1000);   // LED off time
}

Experiments to Try:

  • Fast Blink: Set delay to 100ms - Like a strobe light!
  • Slow Blink: Set delay to 2000ms - A leisurely pace
  • Medium Blink: Set delay to 500ms - Twice per second
Real Arduino: On your actual Arduino, you would change the numbers in the delay() functions and re-upload the code to see the difference.
